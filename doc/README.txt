
	HARD TREES REDO MODULE
	======================

	Prevents digging trees by punching them.

	Version: 0.1.1
	Source code's license: GPLv3
	Textures and locale files' license: CC BY-SA 4.0 International

	Dependencies: default, farming (found in Minetest Game)
	Supported: intllib, fallen_nodes, fallen_trees

	Report bugs or request help on the forum topic:
	https://forum.minetest.net/viewtopic.php?f=9&t=20005


	Installation
	============

	Unzip the archive, rename the folder to hard_trees_redo and place it in
	../minetest/mods/

	If you only want this to be used in a single world, place it in
	../minetest/worlds/WORLD_NAME/worldmods/

	GNU+Linux - If you use a system-wide installation place it in
	~/.minetest/mods/

	For further information or help see:
	https://wiki.minetest.net/Help:Installing_Mods
